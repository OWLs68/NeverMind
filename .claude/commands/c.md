# /c — Короткий % контексту

Виведи **одну стрічку** про поточний % контексту і токени. Без деталей, без breakdown по категоріях, без таблиць — тільки рядок формату:

```
📊 NN% · XXXK/1M
```

**Як виконати:**

1. Знайти файл транскрипту поточної сесії:
   ```bash
   ls -t ~/.claude/projects/*/[a-z0-9]*-*-*-*.jsonl 2>/dev/null | head -1
   ```
2. Запустити lib-скрипт з цим шляхом:
   ```bash
   bash /home/user/NeverMind/.claude/hooks/lib/compute-context-pct.sh <шлях>
   ```
3. Скрипт повертає `<percent> <tokens>` (через пробіл). Сформувати рядок: `📊 ${percent}% · $((tokens / 1000))K/1M`
4. Вивести цей рядок Роману. **Більше нічого** — без пояснень, без рекомендацій, без преамбули.

**Чому коротко:** `/context` (вбудована команда) дає повний breakdown зі списком MCP tools — Роман хоче лише швидку перевірку «скільки залишилось».

**Похибка:** lib читає **останнє** `assistant.message.usage` у транскрипті — реальний контекст для наступного ходу буде на 1-2% більший (твоя репліка + tool results ще не враховані). Це нормально.
