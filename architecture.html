Потік даних — типовий запис

📥 Inbox — єдина точка входу
Користувач вводить: "task купити молоко завтра"
Або без префіксу → AI сам визначає тип
🔍 app-inbox.js — парсинг і роутинг
parseInboxInput() зчитує префікс → направляє в потрібний модуль
⚙️ Domain модуль — обробка
app-tasks.js → processUniversalAction() створює задачу
💾 localStorage — збереження
nm_tasks оновлюється. Відразу — без серверних запитів.
🖥 UI — рендер
Відповідний модуль перемальовує список.
