## 📁 Файлова структура проекту

## 🚀 Як працює деплой

Claude пушить у `claude/**` → auto-merge в `main` → esbuild збирає → GitHub Pages (2-3 хв). Детально → [`docs/TECHNICAL_REFERENCE.md`](docs/TECHNICAL_REFERENCE.md).

**CACHE_NAME у `sw.js`** оновлювати вручну локально перед пушем при зміні коду (формат: `nm-YYYYMMDD-HHMM`). Хук нагадує.

**Екстрений скид** (git reset --hard + push --force) — тільки з явного дозволу Романа. Процедура → [`docs/GIT_EMERGENCY.md`](docs/GIT_EMERGENCY.md).
