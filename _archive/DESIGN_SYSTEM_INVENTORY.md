## Метод збору і обсяг

**3 паралельні агенти просканували:**

1. **`style.css`** — основні CSS-правила і токени у `:root`
2. **`index.html`** — inline стилі у HTML-розмітці (атрибут `style="..."`)
3. **`src/**/*.js`** — inline стилі у JS (`el.style.X=...`, `style="..."` у шаблонах, `.cssText=...`)

**Що НЕ скановано:**
- `bundle.js` (згенерований з `src/`, дублікат)
- `_archive/`, `node_modules/`, `.claude/`
- HTML-файли поза PWA (`test-checklist.html` видалено nudNp, `owl-board-mockup.html` лишився як прототип)

**Що знайдено за обсягом:**
- **30+ унікальних HEX-кодів** у трьох джерелах
- **100+ rgba/rgb варіацій**
- **23 розміри шрифту** (від 8px до 88px)
- **16+ значень `border-radius`** (від 2px до 999px)
- **18+ варіацій `box-shadow`**
- **10 значень `backdrop-filter: blur()`** (від 1px до 32px)
- **10+ часів `transition`** і **10 іменованих `@keyframes`**

**Висновок про обсяг:** дизайн-система фактично існує у **трьох незалежних версіях** (CSS / HTML / JS) і має ~5-7 конфліктів між ними де однакова роль (наприклад «модалка» або «картка») має різні значення.

## Заокруглення

### Розподіл за розмірами

**Малі (2-4px) — для прогресбарів, мінімальних елементів:**
- `2px`, `3px`, `4px`
- Асиметричні чат-бабли: `4px 14px 14px 14px` (від агента) і `14px 4px 14px 14px` (від юзера)

**Малі-середні (6-9px) — кнопки, поля вводу:**
- `6px`, `8px`, `9px`

**Середні (10-14px) — категорії, фільтри:**
- `10px` — категорії, фільтри (часто)
- `11px` — альтернативний
- **`12px`** — НАЙЧАСТІШИЙ (модалки, кнопки, 16+ використань у `finance-modals.js`)
- `13px` — опції уточнення
- `14px` — чат-бабли, асиметричні

**Великі (16-24px) — картки і модалки:**
- `16px` — Tasks/Habits картки у JS
- **`18px`** — CSS стандарт `--card-radius`
- `20px` — теги, alt
- `22px` — cat picker modal
- `24px` — великі модалки Finance (4× використань)

**Дуже великі (28px+):**
- `28px` — надбільші ефекти

**Спеціальні:**
- `50%` — круги (аватари, мікробатони)
- `999px` — pill кнопки

### 🚨 КОНФЛІКТИ BORDER-RADIUS

**1. Картки CSS vs JS — різниця 2px:**
- CSS `--card-radius: 18px`
- Tasks JS `border-radius: 16px`
- Habits JS `border-radius: 16px`

**2. Модалки мають 4 різні радіуси:**
- Finance TX modal `24px`
- Finance Date modal `24px`
- Cat picker `22px`
- Splash `18px`
- Tasks/Habits modal `16px`

## Анімації і переходи

### Transitions у `style.css`

| Тривалість | Кількість | Приклади |
|---|---|---|
| `0.15s` | 5+ | all, opacity, transform, background |
| `0.2s` | 6+ | opacity, background, transform, border-color |
| `0.25s` | 5+ | opacity, transform, font-size, font-weight |
| `0.3s` | 2+ | opacity, transform |
| `0.35s` | 2 | transform (cubic-bezier) |
| `0.4s` | 3+ | background, margin-bottom |
| `0.5s` | 1 | background ease |
| `600ms` | 1 | opacity, transform (ease-out) |

🚨 Дуже близькі значення (`0.15` / `0.2` / `0.25` для подібних дій). Стандартизувати у `--motion-fast` (200ms) / `--motion-normal` (300ms) / `--motion-slow` (500ms).

### Cubic-bezier функції

```
cubic-bezier(0.32, 0.72, 0, 1)         /* плавне замикання */
cubic-bezier(0.34, 1.56, 0.64, 1)      /* spring (tipSlideIn) */
cubic-bezier(0.25, 0.46, 0.45, 0.94)   /* smooth (fadeUp) */
```

|---|---|---|

| `tipSlideIn` | 0.35s spring | слайд з пружиною |

| `fadeUp` | 0.2s/0.35s ease both | затемнення вгору |
| `slideUp` | (не визначена детально) | слайд вгору |
