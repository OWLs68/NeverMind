### B-13 — PWA не оновлюється після 3-5 деплоїв
**Симптом:** застосунок оновлюється нормально кілька разів, потім перестає. Chrome оновлює, PWA — ні.
**Причина:** `cache.addAll()` використовує стандартний fetch який бере файли з HTTP-кешу браузера (GitHub Pages: `max-age=600`). При частих деплоях HTTP-кеш віддає старі файли → SW "оновлюється" але кешує старий контент.
**Фікс:** замінено `cache.addAll()` на індивідуальні `fetch(url, { cache: 'reload' })` — примусово з мережі.
**Статус:** Зафіксовано 02.04.2026
