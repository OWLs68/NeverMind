## 🎯 Базова концепція (не змінювати без усвідомленого рішення)

**Один Inbox — єдине джерело всього.**
Користувач пише що завгодно одним рядком. OWL розбирається сам. Ніякої ручної організації.

**Слоган:** Забий на хаос. Просто роби.

**Головний принцип:** Мінімальне тертя між думкою і дією.

**NeverMind як щоденник:** Кожен запис має timestamp. Повна картина дня — через Вечір і Я.

**Кінцева ціль:** Jarvis — персональний агент який бачить всі сфери життя і діє.

## 🚨 Правила роботи з кодом

1. **БЕЗ "Роби" — нічого не змінювати**
2. **Читай код перед змінами** — завжди, не на пам'ять
3. **Звіряйся з цим файлом** — нові ідеї не повинні суперечити концепції
4. **Не губи фікси** — при кожній зміні перевіряй що попередні фікси збережені
5. **Один баг за раз** — після виправлення запропонуй наступний
6. **Перевіряй синтаксис** — після JS змін: `node --check файл.js`
7. **Mockup перед кодом** — якщо задача про UI
8. **safeAgentReply** — завжди замість прямого `addMsg('agent', reply)`

## 🤖 Принципи OWL

**Єдиний мозок:** `getAIContext()` передається в кожен промпт без винятку.

**Принцип "спочатку збережи, потім уточни":**
1. Одразу зберігає з найімовірнішим варіантом
2. Підтверджує збереження
3. Якщо треба — через 600мс задає ОДНЕ питання (`ask_after`)
4. `clarify` — тільки якщо принципово різні дії (нова звичка vs виконання існуючої)

**Агент працює тільки з реальними даними** — не вигадує ліміти і плани яких немає.

**OWL board — пріоритети:**
1. [КРИТИЧНО] — дедлайн через годину, стрік під загрозою ввечері
2. [ВАЖЛИВО] — прострочені, невиконані звички, бюджет 80%+
3. normal — огляд дня, прогрес, порожній день

**Тихий режим:** 23:00–7:00 — жодних генерацій (фактично 0:00–5:00 активний).

**Антиповтор:** `nm_owl_board_said` — що вже казав сьогодні (скидається щодня).
