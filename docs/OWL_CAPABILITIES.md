# 🦉 Повний список можливостей OWL-агента NeverMind

---

## Дії через чат (actions — команди які AI виконує)

### Створення:
- `create_task` — створити задачу (з кроками, дедлайном, пріоритетом)
- `create_habit` — створити звичку (з днями тижня)
- `create_event` — створити подію в календарі (з датою і часом)
- `create_note` — створити нотатку (з папкою)
- `create_project` — створити проект (+ інтерв'ю для деталей)
- `create_folder` — створити папку нотаток
- `save_finance` — записати витрату або дохід

### Редагування:
- `edit_habit` — змінити назву/дні існуючої звички
- `edit_task` — 🆕 змінити назву/дедлайн/пріоритет задачі
- `update_transaction` — змінити категорію/коментар транзакції
- `update_health_progress` — оновити прогрес лікування
- `update_project_progress` — оновити прогрес проекту
- `update_project_tempo` — змінити темп проекту
- `update_project_risks` — записати ризики проекту

### Завершення:
- `complete_task` — закрити задачу
- `complete_habit` — відмітити звичку
- `complete_step` — відмітити крок задачі
- `complete_project_step` — відмітити крок проекту
- `undo_step` — зняти галочку з кроку
- `add_step` — додати крок до задачі
- `add_project_step` — додати крок до проекту

### Видалення/відновлення:
- `delete_task` — 🆕 видалити задачу через чат
- `delete_habit` — 🆕 видалити звичку через чат
- `delete_folder` — видалити папку з нотатками
- `delete_transaction` — видалити транзакцію
- `restore_deleted` — відновити з кошика (7 днів)
- `reopen_task` — 🆕 перевідкрити закриту задачу
- `move_note` — перемістити нотатку в іншу папку

### Моменти:
- `add_moment` — 🆕 додати момент дня з будь-якої вкладки

### Фінанси (специфічні):
- `set_budget` — встановити бюджет
- `create_category` — створити категорію витрат

### Здоров'я (специфічні):
- `log_health` — записати енергію/сон/біль
- `add_medication` — додати ліки до картки

### Проекти (специфічні):
- `add_project_decision` — записати рішення
- `add_project_metric` — додати метрику
- `add_project_resource` — додати ресурс

### Інше:
- `clarify` — модалка уточнення (коли AI не впевнений)
- `reply` — просто текстова відповідь

## Проактивне табло (board — повідомлення OWL зверху екрану)

### Коли оновлюється:
- При зміні даних — миттєва локальна реакція + AI через 5 сек
- При переключенні вкладки
- При поверненні після 2+ годин відсутності (Welcome Back)
- Кожні 30+ хвилин автоматично
- При першому відвідуванні вкладки — підказка

### Миттєві реакції (без AI, локально):
- Задача закрита → "Зроблено! Одна менше 💪" (5 варіантів)
- Звичка відмічена → "Є! Звичка на місці 🔥" (4 варіанти)
- Quit-звичка → "Тримаєшся! Це головне 💪" (3 варіанти)
- Момент записаний → "Записав момент ✨" (2 варіанти)

### Антиповтор контенту:
- Banned words — заборонені слова з останніх 3 повідомлень
- Similarity filter — відкидає якщо >60% слів збігаються

## Контекст для AI (що AI знає при кожному запиті)

- Дата, час, часовий пояс, день тижня
- Профіль і пам'ять користувача
- 8 активних задач (з кроками, дедлайнами, пріоритетами)
- Нещодавно закриті задачі (24 години)
- Найближчі події та дедлайни (7 днів)
- Всі звички зі статусом (виконано/ні)
- 8 записів Inbox за сьогодні
- Фінансовий контекст (бюджет, витрати)
- Кошик видалених (для відновлення)
- Поточне повідомлення табло
- Настрій дня (якщо обраний)

## Cooldown-система (антиповтор)

| Тема | Інтервал |
|---|---|
| Звичайний пульс | 45 хв |
| Дедлайн через годину | 30 хв |
| Звичка під загрозою ввечері | 1 год |
| Задача застрягла 3+ дні | 6 год |
| Звички не виконані | 3 год |
| Всі звички виконані | 8 год |
| Бюджет 80%+ | 4 год |
| Вечір без підсумку | 4 год |
| Ранковий бриф | 3 год |
| Понеділок — огляд тижня | 6 год |
| П'ятниця — підсумок тижня | 6 год |

## Фази дня

| Фаза | Коли | Роль OWL |
|---|---|---|
| morning | підйом → початок роботи | Надихнути, сфокусувати |
| work | робочий час | Прогрес, м'які нагадування |
| evening | кінець роботи → сон-1г | Підсумки, звички |
| night | перед сном | Тільки критичне |
| silent | сон | OWL мовчить |

## Пам'ять і навчання

- **nm_memory** — AI-профіль користувача (до 50 фактів, оновлюється кожні 3 дні)
- **nm_chip_stats** — які чіпи натискають/ігнорують (50 останніх)
- **nm_recent_actions** — лог дій (20 останніх, для крос-контексту)
- **nm_owl_questions** — анкетування: 12 питань (робота, цілі, інтереси), 1 на день
- **nm_tab_first_visit** — підказки при першому відвідуванні вкладки

---

## Особистість OWL (3 режими)

- **Coach** — прямий, конкретний, підштовхує
- **Partner** — м'який, підтримуючий, друг
- **Mentor** — запитує, направляє, бачить паттерни
