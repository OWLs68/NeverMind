## Модальні вікна (Bottom Sheet)

### Стандартний стиль (затверджено 31.03.2026)

Всі модальні вікна типу "редагування" (звичка, задача, проект, момент тощо) мають однаковий вигляд:

**Зовнішній контейнер:**
```css
position: fixed;
inset: 0;
z-index: 200;
align-items: flex-end;
justify-content: center;
padding: 0 16px 16px;  /* 16px відступ по боках від екрану */
```

**Підложка (backdrop):**
```css
position: absolute;
inset: 0;
background: rgba(0,0,0,0.35);
backdrop-filter: blur(4px);  /* легкий blur всього фону */
```

**Панель модалки:**
```css
background: rgba(255,255,255,0.30);       /* білий, 30% прозорості */
backdrop-filter: blur(32px);              /* сильний blur — фон просвічує */
-webkit-backdrop-filter: blur(32px);
border-radius: 24px;
padding: 24px 24px calc(env(safe-area-inset-bottom)+28px);  /* 24px відступи всередині */
border: 1.5px solid rgba(255,255,255,0.5);
max-height: 80-85vh;
overflow-y: auto;
```

**Handle (риска зверху):**
```css
width: 36px; height: 4px;
background: rgba(0,0,0,0.12);
border-radius: 2px;
margin: 0 auto 18px;
```

### Поля вводу всередині модалки:
```css
background: rgba(255,255,255,0.7);
border: 1.5px solid rgba(30,16,64,0.12);
border-radius: 12px;
padding: 12-13px 16px;
```

### Кнопки внизу модалки:
- **Скасувати:** `flex:1`, `background:rgba(30,16,64,0.06)`, `color:rgba(30,16,64,0.5)`
- **Зберегти:** `flex:2`, акцентний градієнт залежно від вкладки, `color:white`

## Картки задач

- Фон: `linear-gradient(135deg,#c6f3fd,#a8ecfb)` — блакитний градієнт
- Border-radius: `16px`
- Чекбокс виконання: `28x28px`, border-radius `8px`
- Кроки: тап по чекбоксу (< 10px руху) — ставить галочку. Свайп/скрол — ігнорується.

## AI Bar (нижня панель)

- Фон: `rgba(15,10,22,0.72)` — темний напівпрозорий
- Чат-вікно: той самий колір що bar
- Handle (риска): білий `rgba(255,255,255,0.45)` коли чат відкритий, темний коли закритий
- 3 стани: закрито → compact (A) → full (B)
- При переключенні вкладок: стан чату зберігається (не скидається)
