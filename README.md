# NeverMind

Personal PWA productivity agent powered by AI.

**Live:** [owls68.github.io/NeverMind](https://owls68.github.io/NeverMind)

## What is this?

NeverMind is a personal productivity app with an AI agent (OWL) that automatically categorizes your thoughts, tasks, habits, notes, and finances. Write anything — OWL figures out the rest.

- Inbox with AI auto-categorization (GPT-4o-mini)
- Tasks, Habits, Notes, Finance, Health, Projects, Evening review
- Proactive OWL agent — gives advice, asks questions, adapts to your mood
- Calendar with task deadlines
- Works offline (PWA with Service Worker caching)
- No backend, no database — everything in localStorage

## Tech Stack

Vanilla JS, HTML, CSS. No frameworks. Built with esbuild. Deployed on GitHub Pages.
